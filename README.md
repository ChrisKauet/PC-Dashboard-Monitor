# PC Dashboard Monitor

Um dashboard de monitoramento de PC acessível via internet, com integração planejada para Supabase e Vercel.

## Estrutura do Projeto

```
PC-Dashboard-Monitor/
├── client/          # Frontend (React/Vue/etc.)
├── server/          # Backend (Node.js/Python/etc.)
├── docs/            # Documentação
└── scripts/         # Scripts de utilidade
```

## Próximos Passos

1. Configurar coleta de métricas do PC (CPU, RAM, Disco, Rede, etc.)
2. Criar API para exposição das métricas
3. Desenvolver frontend para visualização
4. Integrar com Supabase para armazenamento histórico
5. Deploy no Vercel

## Tecnologias Planejadas

- **Frontend**: React/Vue com charts (Chart.js/D3)
- **Backend**: Node.js/Express ou Python/FastAPI
- **Banco**: Supabase (PostgreSQL)
- **Deploy**: Vercel
- **Coleta de métricas**: Bibliotecas específicas do OS
