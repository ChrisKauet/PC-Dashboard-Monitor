TARGET DESIGN (pc-metrics-dashboard.html):
================================================================================

A clean, light/dark adaptive dashboard with these sections (top to bottom):

1. TOPBAR
   - SYS.MONITOR logo (green dot accent on the dot in "SYS.")
   - Hostname display (right side)
   - Animated pulsing "LIVE" badge (green dot + "LIVE" text, green border)

2. METRICS GRID (4 columns)
   4 cards side by side:
   - CPU    — color #1D9E75 (green),  icon: ti-cpu
   - RAM    — color #534AB7 (purple), icon: ti-database
   - GPU    — color #D85A30 (orange), icon: ti-device-desktop
   - Disco  — color #BA7517 (amber),  icon: ti-server

   Each card contains:
   - Label with Tabler icon (uppercase, letter-spaced)
   - Large mono value + smaller unit (e.g. "67" + "%")
   - Subtitle with detail (e.g. "Intel Core i9 · 16 núcleos")
   - Thin color-coded progress bar (3px height, rounded)
   - 2px top accent line in card's color

3. MID GRID (2fr + 1fr)

   Left card — "Histórico — CPU & RAM":
   - Chart.js line chart, 30 data points history
   - Green (#1D9E75) line for CPU, purple (#534AB7) for RAM
   - Filled area under lines
   - No x-axis labels, y-axis 0-100%
   - Legend: colored squares + labels
   - "Tempo real" green badge

   Right card — "Sistema" info list:
   - Uptime (icon: ti-clock)
   - Download (icon: ti-arrow-down) — MB/s
   - Upload (icon: ti-arrow-up) — MB/s
   - Energia (icon: ti-bolt) — Watts
   - Processos (icon: ti-temperature) — count
   - Each row: icon + label on left, mono value on right
   - "Info" blue badge

4. BOTTOM GRID (1fr + 1fr)

   Left card — "Top processos":
   - Header row: "Processo" | "CPU" | "MEM"
   - 5 rows, each with: colored dot, process name, CPU%, MEM
   - "CPU · MEM" orange badge

   Right card — "Temperaturas":
   - 2×2 grid of temperature items:
     - CPU Package
     - GPU Core
     - Disco SSD
     - Placa-mãe
   - Each: large mono value (°C) + small label below
   - Color thresholds:
     - t-ok   (< 68°C)  → green  (#1D9E75 light / #5DCAA5 dark)
     - t-warm (68-79°C) → amber  (#BA7517 light / #FAC775 dark)
     - t-hot  (≥ 80°C)  → orange (#D85A30 light / #F0997B dark)
   - "Monitorando" orange badge

5. LIGHT/DARK AUTO via prefers-color-scheme media query

6. RESPONSIVE:
   - 4 → 2 columns at ≤ 700px
   - Mid grid and bottom grid stack vertically at ≤ 700px
   - Hostname hidden on mobile (≤ 700px)
   - Metric values shrink at ≤ 420px
